# Foot Base Mesh Generator

MシリーズMac上で動作する、造形用の足ベースメッシュをパラメトリックに生成するPythonアプリです。Webcamは使用せず、PySimpleGUIのスライダー操作からOpenCVの棒状2Dプレビューを更新し、OBJ形式で書き出します。

## セットアップ

Python 3環境を用意し、必要ライブラリをインストールします。

```bash
python3 -m pip install PySimpleGUI opencv-python numpy
```

## 起動方法

```bash
python3 main.py
```

## 操作

- スライダーを動かすと、OpenCVで描画した足の簡易スケルトンがリアルタイムに変化します。
- `右足 / 左足` でX方向ミラーを切り替えます。
- `上面プレビュー / 側面プレビュー / 両方表示` で表示モードを切り替えます。
- `モデル作成` で現在のパラメータから内部メッシュを生成します。
- `OBJ出力` でOBJファイルを書き出します。モデル未作成の場合は自動生成してから保存します。
- `リセット` で標準状態に戻します。

## パラメータ

- 足長: かかとから指先方向までの全体長。
- 足幅: 母趾球から小趾球付近の最大幅。
- 甲の高さ: 足本体上面と足首接続部の高さ。
- かかとの幅: かかと周辺の横幅。
- 土踏まずの高さ: 側面ビューと足裏側メッシュのアーチ量。
- 指の長さ: 親指以外の基準長。
- 親指の長さ: 親指だけの個別長。
- 指の開き: 5本指を扇状に広げる角度。
- 指の曲げ: 指セグメントの屈曲。
- 指の反り: 側面方向の持ち上がり。
- 親指角度: 親指だけの横方向角度。
- 足首角度: 足首点の傾き。
- メッシュ解像度: 足本体グリッド、甲、かかと、指断面の分割数。

## プリセット

- 標準: 一般的な足型。
- 幅広: 足幅と指の開きを大きめにした足型。
- 甲高: 甲の高さと足首接続部を高めにした足型。
- 扁平足気味: 土踏まずを低めにした平坦寄りの足型。
- 指長め: 足指全体を長めにした足型。
- エジプト型: 親指が最も長く、人差し指以降が徐々に短い足型。
- ギリシャ型: 人差し指が最も長く、親指が少し短い足型。
- スクエア型: 親指から中指あたりまでの長さが近い足型。

## OBJ利用想定

出力OBJは、ZBrush、Nomad Sculpt、Blenderなどへ読み込み、Dynamesh、Voxel Remesh、ZRemesher、Sculptなどで後加工するためのベースメッシュです。最初から解剖学的に完全な一体モデルにせず、足本体、甲、かかと、5本指の軽い重なりを許容して、造形しやすい形を優先しています。

OBJグループは以下に分かれます。

- `foot_body`
- `heel`
- `instep`
- `toe_1_big`
- `toe_2`
- `toe_3`
- `toe_4`
- `toe_5`

## ファイル構成

- `main.py`: GUI本体、イベントループ、プレビュー更新、OBJ出力処理。
- `foot_params.py`: dataclassによるパラメータ定義。
- `foot_skeleton.py`: プレビューとメッシュ生成で共通利用するスケルトン計算。
- `foot_preview.py`: OpenCVプレビュー描画とPNGバイト列変換。
- `foot_mesh_generator.py`: 足本体、かかと、甲、足指メッシュ生成。
- `obj_exporter.py`: OBJ出力。
- `presets.py`: 各プリセット値。
